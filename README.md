# Reservation Management System

This project contains a simple reservation management system built with Flask and SQLite.

## Setup
1. Install Python 3.
2. Install dependencies:
   ```bash
   pip install Flask
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Access the reservation form at `http://127.0.0.1:5000/` and the admin interface at `http://127.0.0.1:5000/admin`.
